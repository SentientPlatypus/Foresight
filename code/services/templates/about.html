How it works

Foresight takes financial data from Google Finance. However, the google finance API has been discontinued.
Therefore I needed a solution.
I settled on webscraping.
Foresight is built on several webscraping functions that scrape the data you see from the google finance website. Dont tell them haha. After scraping this data, I display it to you through the flask application (With all the styling and scripts of course) you are viewing right now.
However, Google finance couldnt be the solution to everything. Foresight makes predictions. Tensorflow has the tools for that. Our predictions are done using an LSTM neural network. More on that in a bit.

Made for results

Foresight uses an LSTM neural network to predict next months stock prices. The results are incredible.
